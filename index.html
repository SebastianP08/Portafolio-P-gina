Proyectos en los que he trabajado

“Núcleo Lumínico Orgánico” — Escultura 3D con partículas Este proyecto consiste en la creación de una pieza orgánica desarrollada en Blender. La composición parte de un núcleo esférico central con pequeños núcleos a su alrededor, junto con partículas para enriquecer más la composición.

El proceso empezó a base de una forma esférica a la que se le añadió una estructura de tentáculos y se comenzó a modificar levemente la forma sobre todo del frente de esta.

Mas tarde se añadieron mas esferas alrededor con ligeras modificaciones, repartidas en distintos puntos del viewport para más tarde con la cámara crear una composición a base de la perspectiva.

Después de esto se le integro un material metálico a las esferas para generar un efecto algo realista junto con un Bloom y brillos a la misma, esto además fusionándolo con una luz que a base de “paredes” se puede modificar las zonas que ilumina.

Ahora, para generar un efecto más interesante para el resultado final, se le incluyeron partículas o pequeños puntos luminosos para añadir más dinamismo a la composición.

Para finalizar, usando la perspectiva de la cámara y eligiendo el mejor ángulo, se vería el resultado final.
