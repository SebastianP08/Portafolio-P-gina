Modelador/Escultor 3D

Proceso de creación de personajes Del concepto al modelo final

Este proyecto se centra en el desarrollo de un personaje 3D inspirado en la estética del anime. El objetivo fue explorar la construcción de formas, la estilización y los flujos de trabajo de producción mediante técnicas de blocking, escultura, modelado y retopología.



Planteamiento del Proyecto

El reto de este proyecto consistió en desarrollar un personaje 3D con estética inspirada en anime, basado en un concept art 2D, aplicando un flujo de producción completo. El objetivo principal fue trabajar la forma, el detalle superficial y la optimización del modelo para futuros procesos de animación y renderizado.

Proceso de Desarrollo

Para comenzar el proyecto se implementó un flujo de trabajo estructurado, iniciando con el blocking para establecer las proporciones generales del personajes. Posteriormente se desarrolló el esculpido digital para agregar detalle y definir la anatomía, seguido del desarrollo del cabello y finalmente la optimización del modelo mediante retopología.

Blocking y Definición de Forma

La etapa de blocking permitió establecer las proporciones generales del personajes, definiendo la silueta y la estructura base del modelo. Esta fase fue fundamental para garantizar una correcta base de trabajo antes de iniciar el esculpido detallado

Esculpido Digital y Detalle

Durante la etapa de esculpido se trabajó en el refinamiento de la forma y el desarrollo de detalles superficiales, buscando mantener la estética inspirada en el anime mientras se preservaban volúmenes limpios y coherentes para futuras etapas.

Pelo / Grooming y Retopología

El desarrollo del cabello se realizó buscando un equilibrio entre estilización y control técnico del modelo, permitiendo que la geometría del cabello mantuviera un aspecto visual atractivo sin comprometer el rendimiento del modelo.

La retopología se realizó con el objetivo de optimizar la malla del personaje, reduciendo la complejidad geométrica y preparando el modelo para posibles procesos de animación o render en tiempo real.
